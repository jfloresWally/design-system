Tailwind

Para comenzar a usar los estilos de Wally con Tailwind es necesario descargar el repositorio.

Ir al repositorio en Github ->

Importar directivas y fuentes

Seguidamente debes agregar las directivas de Tailwind y importar la fuente Inter de Google fonts.

                    
                        @import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap');

                        @tailwind base;
                        @tailwind components;
                        @tailwind utilities;

Configurar el tema

¿Cómo agregar el Style Guide a tu proyecto?

Solo tienes que agregar a tu HTML los estilos que te genera Tailwind.

                    
                        <link rel="stylesheet" href="./css/tailwind.css">
