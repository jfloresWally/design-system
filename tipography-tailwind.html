Los estilos de texto de Wally en Tailwind se aplican como utilidades del propio framework. Aquí te dejamos la lista de los estilos de texto más usados.

                    
                        /* Typography class */
                        .text-6xl  -> 60px Bold

¿Cómo aplicar los estilos de texto?

Para aplicar las tipografías de Wally solo es necesario usar las utilidades de Tailwind sobre una etiqueta de texto HTML.

                    
                        <h2 class="text-3xl">Heading 2</h2>

Clases que debe usar

Hay una serie de clases adicionales que tienes que usar para lograr implementar de forma correcta las tipografías de Wally sobre los textos.

.font-bold

Para aplicar un texto en bold.

                    
                        <p class="text-base font-bold">paragraph</p>

.font-inter

Para usar la fuente Inter en tus etiquetas de texto.

                    
                        <p class="text-base font-inter">paragraph</p>
